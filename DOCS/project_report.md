# Project Report

## Project Title

Identification of a De Novo Variant in a Simulated Paediatric Rare Disease Patient Using Trio Whole Exome Sequencing Analysis

---

## Objective

The aim of this project is to identify potential disease-causing de novo variants in a simulated paediatric patient by analysing Whole Exome Sequencing data from a family trio.

---

## Methodology

1. Download sequencing data
2. Perform quality control
3. Trim adapters
4. Align reads to hg38
5. Process BAM files
6. Call variants
7. Perform joint genotyping
8. Filter de novo variants
9. Annotate variants
10. Prioritize candidate genes

---

## Software Used

- FastQC
- Trimmomatic
- BWA-MEM
- SAMtools
- GATK4
- bcftools
- ANNOVAR
- Python

---

## Expected Results

The pipeline identifies candidate de novo variants that may explain the patient's phenotype and produces annotated variant files for clinical interpretation.

---

## Conclusion

Trio Whole Exome Sequencing is a powerful approach for identifying disease-causing variants in rare genetic disorders and supports accurate molecular diagnosis.
